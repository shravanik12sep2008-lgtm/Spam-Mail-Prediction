# Spam Mail Prediction Using Machine Learning

## About Project

This project classifies email messages as Spam or Ham using Machine Learning and Natural Language Processing (NLP). The model is trained using email text data and predicts whether a message is spam or not.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Algorithm

- Logistic Regression
- TF-IDF Vectorizer

## Project Files

- mail_data.csv
- Spam_Mail_Prediction.ipynb
- spam_mail_model.pkl
- tfidf_vectorizer.pkl
- Report.pdf
- Presentation.pptx

## Features

- Data Cleaning
- Text Vectorization
- Model Training
- Spam Detection
- Accuracy Evaluation
